# music-theory-visualizer

## streamlit-apps

The following applications have been deployed to the Streamlit Community Cloud.

- [finderboard](https://music-theory-visualizer-rpse3jrtkbrcxesbw4pu7n.streamlit.app/?default_scale=major)
  - Visualizes scales on the guitar finderboard.
- [n_circles](https://music-theory-visualizer-2hxhgwxc4igdvc4wjt4wzg.streamlit.app/?default_scale=major)
  - Visualizes scales in a style similar to Open Music N-Circle.
